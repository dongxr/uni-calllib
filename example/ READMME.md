# 开发测试说明

## example 项目

### 开发环境初始化


1. 安装uni- imlib,uni-calllib依赖 
      
   npm i @rongcloud/imlib-uni   
   npm i @rongcloud/calllib-uni

2. 使用 Hbuilder 打开该目录

3. 将原生插件放到 nativeplugins 目录下

4. 在 manifest.json -> APP 原生插件配置 -> 本地插件中加入原生插件

5. 运行 -> 运行到手机 -> 制作自定义调试基座

6. 运行 -> 运行到手机 -> 运行基座选择 -> 选择自定义基座
